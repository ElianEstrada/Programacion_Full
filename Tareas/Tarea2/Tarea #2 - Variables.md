# ¿Qué Valor queda almancenado en cada una de las variables?

### Ejemplo:

> ```text
> a = 10
> b = 20
> c = 5
> a = a + 3          -> a = 10 + 3          -> a = 13
> b = b + 4 - a      -> b = 20 + 4 - 13     -> b = 11
> c = a + b + c      -> c = 13 + 11 + 5     -> c = 29
> a = a + a          -> a = 13 + 13         -> a = 26
> b = 4                                     -> b = 4
> c = c + 3 - b + 2  -> c = 29 + 3 - 4 + 2  -> c = 30
> ```
> 
> ##### Respuesta: a = 26, b = 4 y c = 30
